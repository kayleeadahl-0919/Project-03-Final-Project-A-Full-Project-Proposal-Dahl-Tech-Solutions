# Project 03: Final Project - A Full Project Proposal

## Project Overview
**Dahl Tech Solutions Website** is a professional two-page website for a fictional small business providing affordable IT support services for small schools and local businesses. The site demonstrates a complete digital project from planning to deployment, including project management documentation, web development, and GitHub Pages hosting.

This project serves as a portfolio piece showcasing skills in:

- HTML and CSS web development  
- Project planning and scope management  
- Risk analysis using the TAME framework  
- Reflection and iterative improvement
